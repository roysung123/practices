1.影片撥放鈕 :使用windows預設撥放器撥放
2.影片封面   :若每行個數(4)大於1會以縮圖顯示，滑鼠移至上方放大
3.搜尋       :搜尋名子或編號
4.每行個數   :每行顯示資料個數，至少1，至多4
5.顯示個數   :總共顯示幾筆資料，若搜尋(3)內有輸入資料，則是搜尋到的資料數
6.重整       :若搜尋(3)內無輸入資料，則依照顯示個數(5)隨機顯示幾筆資料
              若搜尋(3)內有輸入資料，則依照搜尋(3)內資料進行搜尋
